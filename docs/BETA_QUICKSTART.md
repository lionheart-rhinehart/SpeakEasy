# SpeakEasy Beta - Quick Start Guide

Welcome to the SpeakEasy beta! This guide will get you up and running in 5 minutes.

## System Requirements

- Windows 10/11 (64-bit)
- Microphone
- Internet connection (for transcription)
- OpenAI API key

## Installation

1. Download the installer (`SpeakEasy_x.x.x_x64-setup.exe`)
2. Run the installer and follow the prompts
3. SpeakEasy will launch automatically after installation

## First-Time Setup

### 1. Activate Your License

When you first open SpeakEasy, you'll see the license activation screen:

1. Enter the license key provided to you
2. Click "Activate License"
3. Once activated, you'll see the main app

### 2. Set Up Your OpenAI API Key

SpeakEasy uses OpenAI's Whisper for transcription:

1. Get an API key from [OpenAI Platform](https://platform.openai.com/api-keys)
2. Click the Settings button (gear icon) in SpeakEasy
3. Paste your API key in the "OpenAI API Key" field
4. The key is stored securely in your system's credential manager

## Basic Usage

### Voice-to-Text Transcription

1. Press `Ctrl+Space` to start recording
2. Speak into your microphone
3. Press `Ctrl+Space` again to stop
4. Your transcription is automatically pasted at your cursor

### AI Transform

Transform selected text with voice instructions:

1. Select any text in any application
2. Press and hold `Ctrl+` ` (backtick)
3. While holding, speak your instruction (e.g., "make it more professional")
4. Release the key
5. The transformed text replaces your selection

### Voice Commands

Execute actions by voice:

1. Press and hold `Ctrl+Shift+Space`
2. Say the name of an action (e.g., "open gmail")
3. Release the key
4. The action executes automatically

## Default Hotkeys

| Action | Hotkey |
|--------|--------|
| Record/Transcribe | `Ctrl+Space` |
| AI Transform | `Ctrl+` ` (hold) |
| Voice Commands | `Ctrl+Shift+Space` (hold) |
| History | `Ctrl+H` |

All hotkeys can be customized in Settings.

## Troubleshooting

### "Checking license..." stuck
- Ensure you have an internet connection
- The app needs to validate your license online

### No sound after recording
- Check that your microphone is selected in Settings
- Try a different microphone if available

### Transcription failed
- Verify your OpenAI API key is correct
- Check that you have credits in your OpenAI account
