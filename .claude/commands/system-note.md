# /system-note (Deprecated)

This command has been renamed to `/system-upgrade-push`.

## Why the Rename?

The new naming creates clear symmetry:
- `/system-upgrade-pull` - Pull framework updates FROM High Level Vibing
- `/system-upgrade-push` - Push framework issues TO High Level Vibing

## What to Do

Use `/system-upgrade-push` instead.

```
/system-upgrade-push
```

This is an enhanced version with:
- Rich context scanning from your conversation
- Structured handoff prompts
- Better issue documentation
