# /system-upgrade (Deprecated)

This command has been renamed to `/system-upgrade-pull`.

## Why the Rename?

The new naming creates clear symmetry:
- `/system-upgrade-pull` - Pull framework updates FROM High Level Vibing
- `/system-upgrade-push` - Push framework issues TO High Level Vibing

## What to Do

Use `/system-upgrade-pull` instead.

```
/system-upgrade-pull
```

This is exactly the same command with a clearer name.
